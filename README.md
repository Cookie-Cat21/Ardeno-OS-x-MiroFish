# Ardeno OS x MiroFish: The Unified Agency

A next-generation, 100% free multi-modal AI agency. This monorepo consolidates the Ardeno OS Command Center with the MiroFish Parallel Society engine.

## Repository Structure
- **apps/os**: The Ardeno OS frontend (Vite/React/Supabase).
- **services/mirofish**: The Python-based agent simulation backend.

## One-Click Startup
To boot the entire agency (Frontend + Backend), run:
```powershell
./start-agency.ps1
```
Or use the batch file:
```cmd
start-agency.bat
```

## Features
- **100% Free AI Stack**: Powered by Gemini 2.0 Flash (Free), OpenRouter Free Models, and Pollinations.ai.
- **300+ Specialized Agents**: Persistent departmental workers with evolving wisdom.
- **Multi-Modal Execution**: Real-time generation of UI assets, React code, and Vocal mission briefings.
- **Strategic Foresight**: Proactive simulation of market and technical risks.
